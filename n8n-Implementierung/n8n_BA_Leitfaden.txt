Ich habe das Ziel auf einen testbaren n8n-PoC für den Einkaufsprozess fokussiert:

```bash
docker volume create n8n_data
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -e GENERIC_TIMEZONE="Europe/Berlin" \
  -e TZ="Europe/Berlin" \
  -e N8N_ENFORCE_SETTINGS_FILE_PERMISSIONS=true \
  -e N8N_RUNNERS_ENABLED=true \
  -e N8N_EDITOR_BASE_URL="http://localhost:5678" \
  -v n8n_data:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
```

Dann im Browser: `http://localhost:5678`

### 3.3 Ollama für lokale KI (optional)

**Knoten 7: `AF2-KI-Ergebnis-uebernehmen` (optional, Code)**

Ohne KI verbinde ich Knoten 5 direkt mit Knoten 8.

```javascript
const basis = $('AF2-Validieren-Materialtext').all(0, $runIndex);

function parseKi(rawJson) {
  let raw = rawJson.output ?? rawJson.text ?? rawJson.response ?? rawJson.data ?? rawJson;

  if (typeof raw === 'string') {
    const cleaned = raw
      .replace(/^```json\s*/i, '')
      .replace(/^```/i, '')
      .replace(/```$/i, '')
      .trim();
    try { raw = JSON.parse(cleaned); } catch { return null; }
  }

  if (raw && typeof raw === 'object' && raw.output && typeof raw.output === 'object') {
    raw = raw.output;
  }
  if (!raw || typeof raw !== 'object') return null;

  return {
    materialtyp: String(raw.materialtyp ?? raw.kategorie ?? '').trim(),
    suchbegriffe: Array.isArray(raw.suchbegriffe)
      ? raw.suchbegriffe.map(v => String(v).trim()).filter(Boolean)
      : [],
    confidence: Number(raw.confidence ?? raw.konfidenz ?? 0),
    begruendung: String(raw.begruendung ?? '').trim(),
  };
}

return basis.map((item, idx) => {
  const json = { ...item.json };
  const ki = parseKi(items[idx]?.json ?? {});

  if (ki && ki.materialtyp) {
    json.materialtyp = ki.materialtyp;
    json.suchbegriffe = ki.suchbegriffe;
    json.kategorie = ki.materialtyp;
    json.kategorieQuelle = 'KI';
    json.kategorieConfidence = ki.confidence;
    json.kategorieBegruendung = ki.begruendung;
  } else {
    json.materialtyp = json.Werkstoff || json.Artikelbezeichnung || 'Unklassifiziert';
    json.suchbegriffe = [json.Werkstoff, json.Artikelbezeichnung]
      .filter(Boolean)
      .map(v => String(v).trim());
    json.kategorie = json.materialtyp;
    json.kategorieQuelle = 'Fallback';
    json.kategorieConfidence = ki?.confidence ?? 0;
    json.kategorieBegruendung = ki?.begruendung ?? 'Kein valides KI-Ergebnis';
  }

  return { json };
});
```

### 6.2 Phase 2 – Lieferantenzuordnung

**Knoten 8: `AF3-Lieferantenliste-laden`**
- Read/Write Files from Disk: `lieferanten.csv` lesen
- danach Extract from File: CSV parsen

**Knoten 9: `AF3-Lieferantenzuordnung` (Code)**

```javascript
const lieferanten = $('AF3-Lieferantenliste-laden').all(0, $runIndex).map(i => i.json);

const norm = (v = '') => String(v)
  .toLowerCase()
  .replace(/ae/g, 'a')
  .replace(/oe/g, 'o')
  .replace(/ue/g, 'u')
  .replace(/[^a-z0-9]+/g, ' ')
  .trim();

const tokens = (v = '') => norm(v).split(/\s+/).filter(Boolean);

return items.map(({ json }) => {
  const materialTokens = [
    ...(Array.isArray(json.suchbegriffe) ? json.suchbegriffe : []),
    json.materialtyp || '',
    json.Werkstoff || '',
    json.Artikelbezeichnung || '',
  ].flatMap(tokens);

  const scored = lieferanten.map(l => {
    const sortimentTokens = String(l.Sortiment || '')
      .split(',')
      .map(s => s.trim())
      .flatMap(tokens);

    let score = 0;
    for (const mt of materialTokens) {
      if (sortimentTokens.includes(mt)) score += 2;
      else if (sortimentTokens.some(st => st.includes(mt) || mt.includes(st))) score += 1;
    }
    return { lieferant: l, score };
  });

  const maxScore = Math.max(...scored.map(s => s.score), 0);
  const passende = scored
    .filter(s => s.score > 0 && s.score >= maxScore - 1)
    .map(s => s.lieferant);

  json.moeglicheLieferanten = passende.map(l => l.Lieferant);
  json.matchingScores = scored.map(s => ({ lieferant: s.lieferant.Lieferant, score: s.score }));
  json.zuordnungStatus = passende.length ? 'zugeordnet' : 'kein passender Lieferant';

  if (!passende.length) json.status = 'offen/pruefpflichtig';
  return { json };
});
```

**Knoten 13: `KP1-Entscheidung-verarbeiten` (Code)**

```javascript
const form = $json.data ?? $json;

const entscheidung = String(form.entscheidung ?? '').trim().toLowerCase();
const ablehnungsgrund = String(form.ablehnungsgrund ?? '').trim();
const aenderungenDirekt = String(form.aenderungen ?? '').trim();
const aenderungenKI = String(form.aenderungen_ki ?? '').trim();
const aenderungen = [aenderungenDirekt, aenderungenKI].filter(Boolean).join('\n');

if (!['freigeben', 'ablehnen'].includes(entscheidung)) {
  throw new Error("Ungueltige Entscheidung. Erlaubt: freigeben oder ablehnen.");
}
if (entscheidung === 'ablehnen' && !ablehnungsgrund) {
  throw new Error("Ablehnung ohne Grund ist nicht zulaessig.");
}

const positionen = $('KP1-Uebersicht-vorbereiten').first().json.positionen;

if (aenderungen) {
  const zeilen = aenderungen.split(/\r?\n/).map(z => z.trim()).filter(Boolean);
  for (const zeile of zeilen) {
    const [posRaw, lieferantenRaw] = zeile.split('|');
    if (!posRaw || lieferantenRaw === undefined) {
      throw new Error(`Ungueltiges Format: ${zeile}`);
    }
    const pos = Number.parseInt(posRaw.trim(), 10);
    if (!Number.isInteger(pos) || pos < 1 || pos > positionen.length) {
      throw new Error(`Ungueltige Positionsnummer: ${posRaw}`);
    }
    const lieferanten = lieferantenRaw.split(',').map(v => v.trim()).filter(Boolean);
    positionen[pos - 1].lieferanten = lieferanten;
    positionen[pos - 1].zuordnungStatus = lieferanten.length ? 'zugeordnet' : 'kein passender Lieferant';
  }
}

return positionen.map(p => ({
  json: {
    Artikelbezeichnung: p.artikel,
    Werkstoff: p.werkstoff,
    Norm: p.norm,
    Menge: p.menge,
    status: p.status,
    moeglicheLieferanten: p.lieferanten,
    zuordnungStatus: p.zuordnungStatus,
    kp1_entscheidung: entscheidung,
    kp1_ablehnungsgrund: ablehnungsgrund,
  },
}));
```

return items.map(({ json }) => {
  const abw = json.ab.menge !== json.bestellung.menge
    || json.ab.preis !== json.bestellung.preis;
  json.abStatus = abw ? 'nicht korrekt' : 'korrekt';
  return { json };
});
```

**Knoten 31: `IF-AB-korrekt`**
- Bedingung: `{{ $json.abStatus }} equals korrekt`
- True → Knoten 32
- False → zurück zu Knoten 29 oder Klärungszweig

**Knoten 32: `AF16-Auf-Lieferung-warten`**
- Typ: No Operation
- Ende des PoC-Workflows
